<h1>Simple & Minimalist Calculator 🧮</h1>

Welcome to my humble calculator project! This lightweight web calculator is crafted with HTML, CSS, and JavaScript, designed for simplicity and efficiency.


<h2>Features 🌟</h2>

Basic Operations: Addition, subtraction, multiplication, and division.
Clear Display: A clean interface for a clutter-free experience.
Responsive Design: Works seamlessly on various devices and screen sizes.
